## 一、新建合成
**1.合成：** 将文件素材进行编辑组合
**2.预设推荐：** HDTV 1080 25
**3.首选项**：
```
显示：所有关键帧
导入：序列素材 25
媒体和磁盘缓存：更改默认路径
自动保存：更改保存间隔
```
**4.快捷键**
```
1.调节锚点位置：Y
2.打开合成设置：Ctrl+K
3.更改时间的显示比例：Alt+滚轮
4.控制预览范围：B（begin）、N（end）
5.逐帧预览：PageUp、PageDown
6.删除播放条前/后的素材：'Alt+[' 或 'Alt+]'
7.将素材裁剪：Shift+Ctrl+D
8.等比例缩放关键帧：选中要改变的关键帧+Alt
9.反向关键帧：选中要更改的关键帧-动画-关键帧辅助-关键帧反向
10.预合成：Ctrl+Shift+C
```
**5.图层属性快捷键**
```
1.锚点：A
2.位置：P
3.缩放：S
4.旋转：R
5.不透明度：T
6.查看关键帧：U
```
