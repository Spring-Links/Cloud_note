**二进制 ---> 十进制**
```
以 1101 为例

方法一：
  8位二进制权值依次为：128 64 32 16 8 4 2 1
  补齐二进制数为8位
  0   0  0  0  1 1 0 1
  128 64 32 16 8 4 2 1
  最终结果为：8+4+1 = 13

-----

方法二：
  将二进制数从右向左排列，每个数字乘以2的次方，次方从0开始，结果依次相加
  1*2^0 + 0*2^1 + 1*2^2 + 1*2^3 = 1+4+8 = 13 
```

**十进制 ---> 二进制**
```
以 171 为例

方法：凑值法
  8位二进制权值依次为：128 64 32 16 8 4 2 1
  当十进制数大于权值时就将该位标记为1，并用十进制数减去该权值
  如：171 > 128，将第一位标记为1，171 - 128 = 43
       43 < 64，将第二位标记为0
       43 > 32，将第三位标记为1，43 - 32 = 11
       以此类推，结果为：10101011
```
